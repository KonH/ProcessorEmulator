    Architecture:

    • R0: accumulator register
    • R1-R4: common register

    Commands:

    0001 x^2 RST r[x] = 0
    0010 x^2 INC r[x]++
    0011 x^2 y^2 MOV r[y] = r[x]
    0100 RSTA r[0] = 0
    0101 INCA r[0]++
    0110 x^2 MOVA r[0] = r[x]
    0111 x^4 JMP go to x
    1000 x^2 y^4 JMZ go to y if r[x] == 0
    1001 x^2 y^2 z^4 JME go to z if r[x] == r[y]
